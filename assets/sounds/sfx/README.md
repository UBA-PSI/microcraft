# Sound Effects Setup

## Source

Sound effects are from **Shapeforms Audio Free SFX**:

https://shapeforms.itch.io/shapeforms-audio-free-sfx

These sound effects are **NOT included** in this repository and **cannot be redistributed** under open source licenses.

You must download them separately from the link above.

## Setup Instructions

1. Download the free SFX pack from https://shapeforms.itch.io/shapeforms-audio-free-sfx
2. Extract the archive
3. Copy the `.wav` files **directly** into this `sfx/` folder (no subfolders)
