# Sprites

## Generation

All unit and building sprites were generated using **Google Imagen 3** (Nano Banana Pro).

## Included Sprites

### Units (RGBA with transparency)
- `blue-worker.png` / `red-worker.png`
- `blue-soldier.png` / `red-soldier.png`

### Buildings (RGBA with transparency)
- `blue-base-alpha.png` / `red-base-alpha.png`
- `blue-barracks-alpha.png` / `red-barracks-alpha.png`

### Seasonal Variants
- `blue-base-xmas-*.png` - Christmas themed base variants
- `blue-barracks-xmas.png` - Christmas themed barracks
